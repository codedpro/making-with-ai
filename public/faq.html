Frequently Asked Questions (FAQs)

General

Q1: What is Making With AI?

A1: Making With AI is the place where it aims to use the incredible gift of Artificial Intelligence, and let it help others to unleash their creative ideas to reality quickly and easily.

Q2: How to use the site?

A2: The site allows you to choose what, how, and how many. It allows you to generate any image based on a simple description, how many images (1-5) and in what style to generate the images.

Q3: Is there a limit of how many images the user can generate?

A3: Yes and no. Basically, the user can generate up to 5 images per 60 seconds, and after that, the user can continue to generate another 5 images. The site aims to expand in the future the amount of images it can generate per time.

Q4: Can the generated images be used for commercial purposes?

A4: About commercial usage of generated images, please refer to the Terms of Use page.

Q5: How to contact Making With AI?

A5: For any support or business inquiries, please contact: info@makingwithai.com

Q6: How do I report an inappropriate image?

A6: In case of encountering an inappropriate image, please contact info@makingwithai.com with details.

Technical

Q7: Are there system requirements for using Making With AI?

A7: No. The Making With AI site has no hardware system requirements for both desktop and mobile. The performance experience of the site will depend on the user's internet speed and stability.

Q8: What to do if the image generation fails?

A8: If an image generation fails, an error message should appear. There is always an option to try generating again, but if the issue persists, please contact the support.

Q9: Is it possible to view images in the Gallery in full size?

A9: Yes. Simply click on any image you want to view in full size.

Q10: How to download a generated image?

A10: Click on the generated image, select the desired resolution and format, and click "Download".

Q11: What formats and resolutions are available for download?

A11: The available image formats are:

  • JPG
  • PNG
  • WebP
The available resolutions are:
  • 1024x1024
  • 1792x1024
  • 1024x1792
The site aims to expand in the future the types of formats and resolutions.

Q12: How to change the theme of the site?

A12: Click on the theme icon (half-circle) at the top of the page to select between light or dark themes.

Image Management

Q13: Is it possible to search and filter generated images?

A13: Yes. Use the search box to find images by name and use the filter dropdown to sort images by date or alphabetically.

Q14: How to archive generated images?

A14: Generated images are automatically added to the "Archive" section. The archived images are only saved locally. The site aims in the future to allow users to save their archived images on cloud by creating an account on the site.

Q15: How to delete an image from the archive?

A15: Open the "Archive" section, click on the desired image for deletion, and click the delete icon (Ã—) on the image. Please note that there is no confirmation message before deletion. Deleted images cannot be restored. The site aims in the future to include a confirmation message and an option to restore deleted images.

Q16: Is it possible to change the style of the generated images?

A16: Yes. There are built options for different styles of the generated images, simply select the desired option from the menu.

Gallery

Q17: What is the Gallery?

A17: The Gallery is a source of AI generated images provided by the site that users can explore and download. AI generated images by users won't be published in the gallery or in any other source. The site aims in the future to showcase AI generated images by others.

Q18: How to access the Gallery?

A18: Access the Gallery by clicking on the "Gallery" link in the toolbar at the top of the page.

Q19: Is it possible to search for specific images in the Gallery?

A19: Yes. Simply use the search input at the top of the Gallery page to search for images by name. There is also a filter with options for sorting images.

Q20: What filter options are available in the Gallery?

A20: The filter options are:

  • Latest-Oldest
  • Oldest-Latest
  • A-Z
  • Z-A

Q21: How to download images from the Gallery?

A21: While viewing an image in full size, select the desired format and resolution, then click the "Download" button.
